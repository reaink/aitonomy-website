## Getting Started

First, run the development server:

install pnpm

```bash
corepack enable pnpm
```

or global install pnpm

```bash
npm install -g pnpm
```

install dependencies

```bash
pnpm install
```

set env file

```bash
cp .env.example .env.local
```

run development server

```bash
pnpm dev
```
